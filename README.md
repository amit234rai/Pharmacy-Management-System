# Pharmacy Management System

A C++ console application designed to manage pharmacy operations, including inventory tracking, processing orders, and handling medical data. 

---

## Project Structure

This repository contains the complete source code and the data file required to run the system.

*   **base.cpp** – The entry point of the application; handles the main menu and core logic loop.
*   **pharmacy.cpp & functions.cpp** – Contains core functionalities for managing medicine stock, suppliers, and system configurations.
*   **order.cpp** – Manages customer billing, cart processing, and order history.
*   **data.bin** – A lightweight binary file used by the system to permanently store and retrieve pharmacy data (inventory, records, etc.).
